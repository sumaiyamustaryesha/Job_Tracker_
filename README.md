Q1. What is the difference between getElementById, getElementsByClassName, and querySelector / querySelectorAll?


Ans : 

getElementById → selects one element by id

getElementsByClassName → selects multiple elements by class

querySelector → selects first matching element

querySelectorAll → selects all matching elements

---

Q2. How do you create and insert a new element into the DOM?


Ans : 

Create ~ document.createElement()

Insert ~ appendChild() or append()

---

Q3. What is Event Bubbling? And how does it work?


Ans :

Event moves from child to parent when clicked.

---

Q4. What is Event Delegation in JavaScript? Why is it useful?


Ans :

Adding event listener to parent instead of child elements.

Useful for less code and better performance.

---


Q5. What is the difference between preventDefault() and stopPropagation() methods?


Ans :

preventDefault() → stops default behavior

stopPropagation() → stops event from going to parent
